# Orion v1.1.0 — Search, People, Cast and Constellation

> Draft release notes. v1.1.0 has not been published. The application package remains at v1.0.10 until final user and release validation succeeds.

## Highlights

- Search now retains movies, television, and people with independent type and cinema filters for Global, Hollywood, Bollywood, South Indian, Korean, Japanese, and Chinese results.
- Same-name titles show year, cinema, language, available region, rating, and original-title context.
- Quick Search uses a larger non-blurred workspace with responsive two-column results.
- Added Person pages with biography, facts, Known For, normalized filmography, roles, jobs, and Movie/TV navigation.
- Added cast rails and compact directors, writers, creators, and lead-producer groups to Movie and TV pages.
- Added Constellation, a credit-derived performers-and-creators catalog with regional cinema pools, craft/media filters, personalized local seeds, progressive loading, partial-failure recovery, and a bounded 24-hour cache.
- Constellation Load more retains two-request TMDB concurrency while revealing mapped people as each credit wave completes; valid cached pools no longer remap on every visit.
- Added Online/Offline status beside battery information with measured metadata-service round-trip latency in milliseconds.
- Repaired missing legacy My List metadata and stale manual-order entries without discarding user data.

## Compatibility

No downloader, playback, IPC, preload, credential, secure-storage, or provider contract is removed. Existing v1.0.10 profiles, settings, downloads, partial files, history, progress, subtitles, and secure keys remain compatible.

## Release status

Automated repository and Electron checks are required to pass, followed by user validation, six-theme review, live streaming/downloader smoke tests, copied-profile validation, and clean Windows installer/ZIP testing. Publishing remains intentionally paused until these gates are accepted.
