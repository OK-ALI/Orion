# Orion v1.0.10 — Performance, Windows Integration and Final Visual Polish

v1.0.10 focuses on smooth playback, responsive navigation and consistent interaction design across a wider range of Windows hardware.

## Highlights

- Added invisible Quality, Balanced, and Efficiency adaptation driven by local CPU, memory, playback, battery, and Windows power-pressure signals.
- Added battery percentage and charging state to Orion's title bar and tray, with once-per-cycle alerts at 20% and 10%.
- At critical battery, Orion resumably pauses downloads while playback continues, then resumes only those automatic pauses after AC power returns.
- Added Windows media-session metadata and Bluetooth/media-key support for play, pause, stop, next episode, and previous/restart.
- Bluetooth and keyboard volume controls remain owned by Windows, so they adjust the selected speakers or headphones without double-changing Orion's internal player volume.
- Added local, redacted performance and battery diagnostics in Settings.
- Added a live System Integration section with Windows media-control preferences, media-session status, detected audio-output labels, and a shortcut to Windows sound settings.
- Added YouTube-style playback shortcuts across embedded, mini, pop-out, and local playback, including K, J/L, percentage seeking, captions, playback speed, mini-player, and episode navigation.
- Repaired the missing Playback continuity and Discovery region selectors caused by Settings rows discarding their supplied controls.
- Appearance, battery, and system-integration changes now apply and persist immediately.
- Reduced background animation cost under pressure and retained the v1.0.9 fullscreen ambient/Orbit safeguards.
- Rebuilt the complete mini-player shell as a true 16:9 surface from 320–640px wide, with saved width/position migration and a clipped 16px frame.
- Replaced the mini-player header with an auto-hiding in-video toolbar that remains available during pointer, touch, keyboard, drag and resize interaction.
- Added Home-carousel hover headroom so lifted posters, borders and shadows no longer clip at the top or rail edges.
- Added live Interaction appearance settings for Subtle, Balanced and Vivid hover behavior, automatic or custom hover color, glow strength and reset-to-theme defaults.
- Added a themed interaction preview and synchronized mini-player, Downloads, local-player and shortcut-dialog surfaces through semantic design tokens.
- Added a theme-color regression gate for newly tokenized surfaces and backup support for all new interaction preferences.

## Compatibility

Existing settings, secure provider keys, downloads, partial files, history, and playback progress remain compatible. Orion does not enumerate or pair Bluetooth devices; connect them normally through Windows before launching playback.
