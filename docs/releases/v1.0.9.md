# Orion v1.0.9 release checklist

## Scope

Cosmic Polish and Playback Continuity: scoped stream detection, richer Downloads, restricted local playback, unified mini/pop-out handoffs, dynamic ambient palettes, Orion visual identity, discovery hubs, and Library/Continue polish.

## Automated gates

- [x] `npm.cmd run check`
- [x] `npm.cmd run test:electron`
- [x] No hand-written source file exceeds 800 lines.
- [x] IPC fixture includes the additive v1.0.9 channels and all v1.0.8 aliases.
- [x] Secret scan contains no TMDB, SubDL, Wyzie, cookies, or captured request tokens.

## Upgrade and live smoke tests

- [ ] A copied v1.0.8 profile retains settings, secure keys, history, progress, downloads, subtitles, and partial jobs.
- [ ] Existing HLS protected-host downloads still complete through the proven strategy order.
- [ ] Late modal opening, source switching, extensionless HLS, DASH, direct video, and unsupported browser-only messaging behave correctly.
- [ ] Local MP4 range seeking, subtitles, missing files, external open, and Show in folder work.
- [ ] Embedded → mini → pop-out → mini handoffs resume within ±2 seconds without duplicate audio.
- [ ] Ambient sampling stops when paused/hidden/minimized, under Reduced Motion, and on battery.

## Visual and package tests

- [ ] Home, Discover, Downloads, Library, Settings, Movie, TV, mini-player, and local player checked in Cosmic and Orion Day.
- [ ] Keyboard focus, 200% zoom, compact windows, and Reduced Motion checked.
- [ ] NSIS installer and ZIP start on a clean Windows machine without Python, system yt-dlp, or ffmpeg.
- [ ] GitHub release is created only after all required boxes pass.
